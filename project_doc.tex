\textbf{AI Blog Writing Agent -- Multi-Agent Research \& Content Generation System} \hfill \href{https://github.com/AjAyrAo43/AI-BLOG-WRITING-AGENT}{GitHub} \\
\textit{Tech Stack: Python, LangGraph, LangChain, FastAPI, Groq API, Tavily Search, Docker, AWS (ECR/EC2), GitHub Actions}

\begin{itemize}
    \item Architected a \textbf{multi-agent system} using \textbf{LangGraph} for automated web research and structured blog generation, featuring a routing node that autonomously decides when external data is required.
    \item Optimized performance and reduced latency by switching to \textbf{Llama 3.1 8B (Groq)}, implementing robust boolean validation and context window management to handle large research payloads.
    \item Developed a modern, responsive frontend using \textbf{Vanilla JavaScript} and \textbf{CSS}, featuring a mobile-friendly dashboard for real-time generation tracking and markdown preview.
    \item Containerized the application with \textbf{Docker} and established an automated \textbf{CI/CD pipeline} via \textbf{GitHub Actions} for seamless deployment to \textbf{AWS ECR} and \textbf{EC2}.
\end{itemize}
